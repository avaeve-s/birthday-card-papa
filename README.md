# การ์ดวันเกิดออนไลน์ 💌

การ์ดวันเกิดแบบ vintage letter (โทนน้ำตาล-ครีม) คลิกที่ซองจดหมายเพื่อเปิดอ่านคำอวยพร มี confetti และเพลงคลอ

## วิธีใช้งานเบื้องต้น

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม

## การใส่รูปภาพ

การ์ดจะแสดงรูปสไตล์โพลารอยด์ 2 รูปตอนเปิดจดหมาย ให้เตรียมไฟล์รูป 2 ไฟล์ ชื่อ:

- `photo1.jpg`
- `photo2.jpg`

วางไว้โฟลเดอร์เดียวกับ `index.html` (โฟลเดอร์เดียวกับที่มี `music.mp3`)

- แนะนำรูปแนวตั้งหรือสี่เหลี่ยมจัตุรัส เพื่อให้ครอปสวยในกรอบโพลารอยด์
- ถ้าไม่มีไฟล์รูป กรอบโพลารอยด์จะยังโชว์อยู่แต่จะเป็นช่องว่าง แนะนำให้ใส่รูปก่อนอัพขึ้น GitHub

## การใส่เพลงประกอบ

ไฟล์นี้จะเล่นเพลงจากไฟล์ชื่อ `music.mp3` ที่อยู่โฟลเดอร์เดียวกับ `index.html`
เนื่องจากเรื่องลิขสิทธิ์เพลง คุณต้องหาไฟล์เพลงมาใส่เอง แนะนำแหล่งเพลงฟรีลิขสิทธิ์ (royalty-free):

- Pixabay Music: https://pixabay.com/music/
- YouTube Audio Library
- Free Music Archive

เลือกเพลงแนว acoustic / lo-fi / piano เบาๆ แล้ว rename ไฟล์เป็น `music.mp3` วางไว้โฟลเดอร์เดียวกับ `index.html`

หากไม่มีไฟล์เพลง การ์ดจะยังใช้งานได้ปกติ แค่จะไม่มีเสียงเพลง

## วิธีอัพโหลดขึ้น GitHub Pages (ฟรี)

1. สร้าง repository ใหม่บน GitHub เช่น `birthday-card-papa`
2. อัพโหลดไฟล์ `index.html` (และ `music.mp3` ถ้ามี) ขึ้น repository
3. เข้าไปที่ **Settings > Pages**
4. ในหัวข้อ **Branch** เลือก `main` และโฟลเดอร์ `/ (root)` แล้วกด **Save**
5. รอสักครู่ GitHub จะสร้างลิงก์ให้ เช่น
   `https://ชื่อผู้ใช้.github.io/birthday-card-papa/`
6. คัดลอกลิงก์นี้ไปแชร์ให้ป๊าได้เลย 🎉

## ปรับแต่งข้อความ/สี

- ข้อความคำอวยพรอยู่ใน `<div class="letter-body">` ในไฟล์ `index.html`
- เปลี่ยนสีธีมได้ที่ตัวแปรด้านบนของไฟล์ในส่วน `:root { ... }`
- ชื่อผู้ส่งอยู่ใน `<div class="signature">— อีฟ</div>`
